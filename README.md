# 微博爬虫
目前是以一个人为基础，爬取所有微博，并做分析，后期争取能自动分析

## 已完成工作

1. 爬取个人所有微博博文，图片链接及各种信息
2. 提取信息
3. 需要的作图函数已完成
4. 自动根据输入生成等下载url文件，下载错误url记录，成功下载url记录

## 当前工作

1. 提取已爬取的信息，并自动作图
2. 爬取评论及信息
3. 图片自动下载程序
4. 长文自动下载
5. 断点续传（包括爬微博，下图片，下长文的）
