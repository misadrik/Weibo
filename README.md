# 微博爬虫
目前是以一个人为基础，爬取所有微博，并做分析

## 已完成工作

1. 爬取个人所有微博博文，图片链接及各种信息
2. 提取信息
3. 需要的作图函数已完成

## 当前工作

1. 提取已爬取的信息，并自动作图
2. 爬取评论及信息
3. 图片自动下载程序
4. 长文自动下载

## 未来规划

1. 断点续传（包括爬微博，下图片，下长文的）
2. 多进程爬取
3. 加代理爬取
4. 争取只要一个id 或加上cookie就可以用于分析一个人的微博数据
